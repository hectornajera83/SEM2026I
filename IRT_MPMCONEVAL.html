Segundo ejemplo

Ahora vamos hacer estimaciones de TRI considerando el ndice de seis carencias del CONEVAL. Para ello utilizaremos los datos 2018.

library(haven)
D18<-read_dta("pobreza_18sample.dta")

El modelo del CONEVAL no parece pensar a las carencias en trminos exclusivos de diferencias en severidad. Noten que tambin se pueden pasar los nombres de las variables. Con la funcin coef() se extraen los valores de la discriminacin a y de la severidad b. Hay valores un tanto bajos: ic_rezedu, ic_asalud, ic_ali.

m3<-mirt(D18[,c("ic_rezedu", "ic_asalud", "ic_segsoc", "ic_sbv", "ic_ali", "ic_cv")], 1, itemtype = '2PL', weights=D18$factor)

Aplicando los anlisis avanzados al modelo CONEVAL

Veamos cmo se comportan estos mismos anlisis con el ndice de carencias del CONEVAL:

# Anlisis rpido de la relacin score-theta para CONEVAL
cat("\n=== ANLISIS CONEVAL: RELACIN SCORE-THETA ===\n")
## 
## === ANLISIS CONEVAL: RELACIN SCORE-THETA ===
# Calcular scores y thetas
scores_coneval <- rowSums(D18[,c("ic_rezedu", "ic_asalud", "ic_segsoc", "ic_sbv", "ic_ali", "ic_cv")])
theta_coneval_matrix <- fscores(m3, method = "EAP", full.scores = TRUE)

# Extraer solo los valores de theta
if(is.matrix(theta_coneval_matrix)) {
  theta_coneval <- theta_coneval_matrix[,1]
} else {
  theta_coneval <- as.vector(theta_coneval_matrix)
}

df_coneval <- data.frame(
  score = scores_coneval,
  theta = theta_coneval
)

# Tabla de conversin
tabla_coneval <- aggregate(theta ~ score, df_coneval, 
                          function(x) c(mean = round(mean(x), 3),
                                      n = length(x)))
tabla_coneval <- do.call(data.frame, tabla_coneval)
colnames(tabla_coneval) <- c("Num_Carencias", "Theta_Medio", "N")

print(tabla_coneval)

Extras: Estimacin del ajuste de cada indicador.

La funcin itemfit muestra los cambios en \(\chi^2\) atribuibles a los indicadores y el valor de RMSEA por indicador. Estos estadsticos pueden usarse para identificar items problemticos en el contexto del modelo estimado. Valores de RMSEA.S_X2>.05 indican un pobre ajuste.
